ApplyLedgerRow

Member resides in the imsAO_AR_Application Class

Summary


Called from the AutoApply routine on the Ledger object to distribute cash. The caller has taken care of instantiating this object and deciding what ledger row to apply the cash to.

Remarks


This is easiest to do inside the application object (instead of the ledger) because we can use the internal change flag.

Returns


Returns the amount applied.
