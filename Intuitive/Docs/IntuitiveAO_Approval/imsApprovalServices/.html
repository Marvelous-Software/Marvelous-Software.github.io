Member resides in the imsApprovalServices Class

Summary


Services layer for approvals.

Remarks


This class uses the strategy pattern to handle document specific approvals. The class imsApprovalDocumentType contains all of the properties/methods necessary to handle document specific logic for approvals.Mod developers that are adding a new approval document type should register a custom imsApprovalDocumentType by instantiating the class with the appropriate parameters in the constructor and calling imsApprovalServices.RegisterApprovalDocumentType().
