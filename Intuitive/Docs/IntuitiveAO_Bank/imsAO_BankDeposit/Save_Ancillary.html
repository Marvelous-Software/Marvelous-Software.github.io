Save_Ancillary

Member resides in the imsAO_BankDeposit Class

Summary


Save the Secondary ledger objects in the cache.

Remarks


This routine is a place to save code that deals with anciliary domains. Data that is to be saved as part of the Save transaction should be added to the SQL and/or TQ_Other collections of R2DT. The routine is executed only if ProcessingNow = True.

Returns


Return Data.
