RoundUpValue

Member resides in the imsAO_BidMatrixDetail Class

Summary


Rounds the value based on the Unit Of Measure. If the unit of measure is marked as round up the value is rounded to the next larger integer. If not the value is rounded to 4 decimal places.

Remarks


If the value does not have any decimal places, the value is returned.

Returns


The value after round up by ao IntuitiveAO_Item.imsAO_UnitOfMeasure.
