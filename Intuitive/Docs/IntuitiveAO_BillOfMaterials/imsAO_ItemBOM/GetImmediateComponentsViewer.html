GetImmediateComponentsViewer

Member resides in the imsAO_ItemBOM Class

Summary


These routines call special UDFs that return single levels but flag the rows so we can see when to explode further.

Remarks


They are used to fill treeview nodes, level by level as the user explodes into the BOM or WU gradually This is far superior to loading the entire bom and not being able to show when there are lower/higher items. Extra fields beyond PST are defined in the UDFs

Returns


Returns a dataset containing PST records for the parent item passed in
