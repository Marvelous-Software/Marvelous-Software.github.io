whereami

Member resides in the imsAO_ItemBOM Class

Summary


Determines where date fits within the specified range, Returns integer

Remarks


compares a date with a date range and determines if it is before, during or after the range returns 1 if eff date before begin date returns 2 if eff date after begin date but before end date returns 3 if eff date is after end date returns -1 if all cases fail

Returns


integer
