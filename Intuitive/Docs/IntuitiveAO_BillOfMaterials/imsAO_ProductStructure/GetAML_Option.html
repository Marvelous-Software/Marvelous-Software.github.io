GetAML_Option

Member resides in the imsAO_ProductStructure Class

Summary


Determine the AML Item to replace the AML Generic.

Returns


Returns the customer preferred mfg. when marked as 'ok to use'. When the preferred mfg. is 'not ok to' use return the highest priority AML option that is 'ok to use'. Guid.Empty is returned when there is not an AML Option that is marked 'ok to use'.
