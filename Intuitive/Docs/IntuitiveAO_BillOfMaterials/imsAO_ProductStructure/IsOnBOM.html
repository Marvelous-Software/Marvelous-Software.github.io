IsOnBOM

Member resides in the imsAO_ProductStructure Class

Summary


Returns Boolean based on use of item on a BOM. If UseEffDate is true will use either the date sent in EffDate or Today as a filter on the Stop date. The Start date not filtered so future starts will return a true

Remarks


Checks the supplied value of IMA_RecordID and returns True if it is on a BOM, or if useeffdate is true, returns True if it is on a BOM with a current or future effectivity date false if not on a BOM or not on a BOM with a current or future effectivity date uses the passed value for EffDate or Today if nothing passed

Returns


Boolean
