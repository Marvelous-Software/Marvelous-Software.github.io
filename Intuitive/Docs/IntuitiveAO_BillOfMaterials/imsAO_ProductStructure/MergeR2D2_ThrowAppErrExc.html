MergeR2D2_ThrowAppErrExc

Member resides in the imsAO_ProductStructure Class

Summary


Merges the ReturnData and Throws an imsxUnexpectedException if AnyAppErrs.

Remarks


Intended for use from methods that do SetValue changes but don't return a returnData, such as LoadExiting(). If any app errors occur in this case, it is a developer error and should be corrected.
