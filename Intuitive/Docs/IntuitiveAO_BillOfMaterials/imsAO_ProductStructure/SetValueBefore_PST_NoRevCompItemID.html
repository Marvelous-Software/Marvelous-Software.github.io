SetValueBefore_PST_NoRevCompItemID

Member resides in the imsAO_ProductStructure Class

Summary


Each time the value for NoRevCompItemID is set it must be tested against the parent

Remarks


tests the no rev item id passed in fieldvalue with the no rev item id for the parent if they are the same, an error is returned and one of two things happens. if this is a new row, it is deleted, if not the old value is restored
