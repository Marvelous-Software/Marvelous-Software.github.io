CheckItemQuoteType

Member resides in the imsAO_ProductStructureHeader Class

Summary


Checks the specified IMA_RecordID to see if it is a QuoteItem and if so is the BOMType Quoting.

Remarks


This function will look at the Item table in the IMA_QuoteTypeFlag field to see if it is set. If the flag is set, the function will check that the passed BOMType is "Quoting".

Returns


Boolean
