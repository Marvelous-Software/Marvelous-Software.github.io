CreateDocument_TemplateMailMergeSourceSQL

Member resides in the imsAO_CRMNote Class

Summary


This method must return SQL which will result in a resultset that includes all CRMCompany, CRMCompLocation and CRMContact fields, as well as the EMP_FirstName and EMP_LastName fields from EMP. Even if the data doesn't exist / apply, this resulting SQL must include these fields (even if NULL), otherwise the MailMerge will fail.
