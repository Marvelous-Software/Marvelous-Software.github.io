GetDefaultPriorityPoints

Member resides in the imsAO_CTPDefaultPriorityPoints Class

Summary


This function will query CTPDefaultPriorityPoints table return a hashtable where the hashtable key is CDPP_Key and CDPP_RecordType and the hashtable value is CDPP_Points value for that CDPP_Key value. Or, if no record type, then the CDPP_Key is the hashkey
