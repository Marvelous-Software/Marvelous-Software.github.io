LoadFromTrans_XLoc

Member resides in the imsAO_BillTo_Collection Class

Summary


The base calls this routine if it finds an XLOC attribute in the bill-to transaction header. The value of the attribute is passed in. The routine handles a value of DEFAULT, which means to update the default bill-to object (which is created in the NewSetup of the customer, therefore the problem and the need for this routine.
