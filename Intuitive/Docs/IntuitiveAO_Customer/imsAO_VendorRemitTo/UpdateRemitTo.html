UpdateRemitTo

Member resides in the imsAO_VendorRemitTo Class

Summary


Update the remit to field values to the vendor field values.

Remarks


The value for the remit to field is update when null or the same as the original vendor field. String fields that have different cases are not updated. When the app object is marked as same a corp address the address, city, state, postal code, and country are always updated.
