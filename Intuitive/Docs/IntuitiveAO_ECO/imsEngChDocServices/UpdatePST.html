UpdatePST

Member resides in the imsEngChDocServices Class

Summary


Updates the imsAO_EngChDoc_PST.EngChFieldValue_Collection based on the product structure AO sent in.

Remarks


The update is based on the changes found in the ProductStructure AO. Fields that are different than their original value are stored in the EngChFieldValue_Collection. If a field is in the EngChFieldValue_Collection but is not changed in the ProductStructure AO, the field will be deleted.The list of supported fields is limitied.
