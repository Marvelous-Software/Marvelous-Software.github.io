SetValue_WLL_MWH_ID

Member resides in the imsAO_WhsLocLotTransaction Class

Summary


Validates, updates the value of and fires off dependent changes for field WLL_MWH_ID.

Remarks


SetValue_WLL_MWH_ID validates that the FieldValue passed in corresponds to the primary key of an existing object in the InventoryTransaction's MasterWarehouse cache. The matching imsAO_MasterWarehouse object's MWH_WhsName value is then used to set WLL_WarehouseName if it is not already equal.
