SetValue_WLL_WarehouseName

Member resides in the imsAO_WhsLocLotTransaction Class

Summary


Validates, updates the value of and fires off dependent changes for field WLL_WarehouseName.

Remarks


SetValue_WLL_WarehouseName tries to validate the warehouse name by trying to find an object in the InventoryTransaction's MasterWarehouse cache with the same name. If it finds a matching imsAO_MasterWarehouse object in the cache then it sets the WLL_WarehouseName to the passed in FieldValue and sets WLL_MWH_ID to the the found master warehouse object's MWH_ID value.
