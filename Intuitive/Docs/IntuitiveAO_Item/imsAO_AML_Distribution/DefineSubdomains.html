DefineSubdomains

Member resides in the imsAO_AML_Distribution Class

Summary


Define the Subdomains for this level and parameters about them.

Remarks


Subdomains are children app objects to this app object. A such they are collection class objects; peers are not handled here. This code creates a collection of the Subdomain collection classes so that functions such as Load and Save can handle the kids polymorphically. One Add is placed for each Subdomain.
