Copy_RoutHrsExcptFlag

Member resides in the imsAO_Item Class

Summary


The item is required to have a routing record before setting the value to true.

Remarks


When coping routings from the source item, allow the user to set the value to true if the value is true for the source item. When the routing records are not copied leave the value as false and add an information message: The Routing Exception Flag cannot be set for items without routings.
