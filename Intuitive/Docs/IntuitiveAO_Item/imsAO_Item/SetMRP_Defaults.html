SetMRP_Defaults

Member resides in the imsAO_Item Class

Summary


Update the planning flags based on the lead-time code.

Remarks


When the lead-time code is purchased IMA_MakeOnAssyFlag, IMA_WIPInspFlag, IMA_FwdSchedFlag, IMA_BackFlushMaterial, IMA_BackflushLabor, IMA_BackFlushOperationSeq, and IMA_RoutHrsExcptFlag are updated to false. When the lead-time code is manufactured IMA_WIPInspFlag, IMA_FwdSchedFlag, IMA_BackFlushMaterial, IMA_BackflushLabor, IMA_BackFlushOperationSeq, and IMA_RoutHrsExcptFlag are updated to settings entered in the master controls.
