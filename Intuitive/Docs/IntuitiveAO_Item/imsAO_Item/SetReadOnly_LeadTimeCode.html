SetReadOnly_LeadTimeCode

Member resides in the imsAO_Item Class

Summary


Set item field to read-only based on the lead time.

Remarks


When this is a purchased item the following field values are update to false and set the read-only: IMA_MakeOnAssyFlag, IMA_WIPInspFlag, IMA_FwdSchedFlag, IMA_BackFlushMaterial, IMA_BackflushLabor, and IMA_BackFlushOperationSeq. Material cost is only editable when the lead time is purchased.
