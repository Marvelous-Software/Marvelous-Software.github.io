SetValue_AverageCost_After

Member resides in the imsAO_ItemCost Class

Summary


When the Item Cost Type is Average, update the accumulated costs.

Remarks


fits 74000 If the Costing method is changed to either average or last, the user should not be prompted asking if they want to update the accounting value, it should be changed. Average/last costing will not work properly if the accounting value doesn't equal the average/last.
