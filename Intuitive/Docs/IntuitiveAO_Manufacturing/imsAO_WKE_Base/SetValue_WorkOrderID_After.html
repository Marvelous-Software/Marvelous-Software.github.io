SetValue_WorkOrderID_After

Member resides in the imsAO_WKE_Base Class

Summary


Updates l_ItemID, l_FixBurdRate, l_VarBurdRate, WKE_RoutSeqID and the total quantity fields.

Remarks


The variable mOnlyOpSeqID is set when the Work Order ID is validated if only one OpSeqID is open for the work order; if there are more than one it is set to "".
