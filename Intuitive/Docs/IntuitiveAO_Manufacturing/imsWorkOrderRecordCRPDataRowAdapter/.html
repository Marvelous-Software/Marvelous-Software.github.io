Member resides in the imsWorkOrderRecordCRPDataRowAdapter Class

Summary


Adapts a DataRow to the imsIWorkOrderRecord interface for use by CRP.

Remarks


Assumptions: 1. EIAWorkOrderRow has all the fields in EIA. 2. EIA_Ref1 contains the WorkOrder ID. 3. The WOOTable is a part of the cache that will be saved. 4. WOOTable has a relation called WOO_WCL_RELATION to the WCL table in the cache.
