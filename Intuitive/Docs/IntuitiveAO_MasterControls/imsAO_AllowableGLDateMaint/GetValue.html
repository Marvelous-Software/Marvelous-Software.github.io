GetValue

Member resides in the imsAO_AllowableGLDateMaint Class

Summary


Gets a value from the app object. The derived class should not call the base if it determines the field's value (such as with a logical field) else the base needs to be called. Refer to the AO FX Doc for examples and patterns.

Remarks


Three possible sources for a value: database field, logical field, and database field via a join.

Returns


The field value. It will be boxed typed per the DB Type cast routine.
