ListForComboBox

Member resides in the imsAO_ListNonZeroLotQtys Class

Summary


This is a front-end to the List method that tailors the returned dataset more in-line with what a combo box would require.

Remarks


The actual list should always by done by the list function. This method should just preps data -- ususally by restricting the columns returned -- as it does the intended purpose of the call is to fill a combo box.

Returns


A dataset.
