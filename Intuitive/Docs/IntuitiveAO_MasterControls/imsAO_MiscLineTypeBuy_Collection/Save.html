Save

Member resides in the imsAO_MiscLineTypeBuy_Collection Class

Summary


Persists document and changes to the database or the transaction queue.

Remarks


Following Save the app object is marked terminated. R2DT is used to the accumulte transactions. R2D2 is used for the results of the last call so that if an error occurs just the results of that call are returned and not the cummulative results as well. If the Save is a success then R2DT is passed back.

Returns


imsReturnData -- any and all collections possible.
