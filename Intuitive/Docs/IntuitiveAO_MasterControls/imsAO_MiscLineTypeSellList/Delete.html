Delete

Member resides in the imsAO_MiscLineTypeSellList Class

Summary


Flags this object for delete. The routine calls DeleteCheck to see if there are any issues regarding deleting this object or any of its children. If the object is not to be deleted, an app error should be returned. Overriding methods need to call the base routine so that the proper internal flags are set.

Returns


imsReturnData: App Errors, InfoMsgs
