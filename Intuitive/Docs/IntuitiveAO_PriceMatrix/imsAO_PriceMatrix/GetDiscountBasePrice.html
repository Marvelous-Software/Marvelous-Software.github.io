GetDiscountBasePrice

Member resides in the imsAO_PriceMatrix Class

Summary


Gets the base price to use to calculate the Price when using a discount.

Remarks


This expects the CurrencyMultipliers to be previously set and uses them for the calculation.

Returns


If PriceMatrixIsForSpecificItemID or PRM_BasePrice = 0, the IMA_Price. Otherwise, the PRM_BasePrice. It will be converted based on the settings to PriceMatrix.DomesticCurrencyMultiplier And PriceMatrix.PriceMatrixCurrencyMultiplier.
