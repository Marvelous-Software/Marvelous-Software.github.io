GetMatrix

Member resides in the imsAO_PriceMatrix Class

Summary


Returns a table of PriceBreaks for the PriceMatrix for the parameters based on the defined hierarchy.

Remarks


The returned DataTable contains these columns: 1. FinalPrice - Decimal. Contains the correct calculate/converted price to be used for that price break. 2. PRD_StartQty - Double. StartQty of the PriceBreak. 3. PRD_StopQty - Double. StopQty of the PriceBreak. 4. IMA_Price - Decimal. The currency-converted IMA_Price. 5. PRM_BasePrice - Decimal. The currency-converted BasePrice used for discounts, when filled in. 6. PRD_UnitSalesPrice - Decimal. Currency-converted UnitSalesPrice. 7. PRD_DiscountAmount - Decimal. Currency-converted DiscountAmt. 8. PRD_DiscountPercent - Double. 9. PRD_MarkupAmount - Decimal. Currency-converted Markup Amt. 10.PRD_MarkupPercent - Double Price Matrix Hierarchy Order: 1) CustomerID, SoldTo, ItemID, Currency 2) CustomerID, SoldTo, Product Family, Currency 3) CustomerID, SoldTo, All Items, Currency 4) CustomerID, All SoldTos, ItemID, Currency 5) CustomerID, All SoldTos, Product Family, Currency 6) CustomerID, All SoldTos, All Items, Currency 7) Customer Type, All SoldTos, ItemID, Currency 8) Customer Type, All SoldTos, Product Family, Currency 9) Customer Type, All SoldTos, All Items, Currency 10) All Customers, All SoldTos, ItemID, Currency 11) All Customers, All SoldTos, Product Family, Currency 12) All Customers, All SoldTos, All Items, Currency If no PriceMatrix is found, and Not (Currency = DomesticCurrency) and there is a PriceMatrix for the DomesticCurrency with ConvertToForeignCurrency=True, then the DomesticCurrency PriceMatrix is returned.

Returns


An r2d2 with possible app errors. Also, Matrix gets filled with a pricematrix.
