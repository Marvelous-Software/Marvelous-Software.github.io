GetPrice

Member resides in the imsAO_PriceMatrix Class

Summary


Use this override when you don't have seperate line and delivery quantities. Gets the unit price for a specific customer, item, etc.

Remarks


Price Matrix Hierarchy Order: 1) CustomerID, SoldTo, ItemID, Currency 2) CustomerID, SoldTo, Product Family, Currency 3) CustomerID, SoldTo, All Items, Currency 4) CustomerID, All SoldTos, ItemID, Currency 5) CustomerID, All SoldTos, Product Family, Currency 6) CustomerID, All SoldTos, All Items, Currency 7) Customer Type, All SoldTos, ItemID, Currency 8) Customer Type, All SoldTos, Product Family, Currency 9) Customer Type, All SoldTos, All Items, Currency 10) All Customers, All SoldTos, ItemID, Currency 11) All Customers, All SoldTos, Product Family, Currency 12) All Customers, All SoldTos, All Items, Currency If no PriceMatrix is found, and Not (Currency = DomesticCurrency) and there is a PriceMatrix for the DomesticCurrency with ConvertToForeignCurrency=True, then the DomesticCurrency PriceMatrix is returned.

Returns


An r2d2 containing any app errors that occur. Price gets set as the first parameter.
