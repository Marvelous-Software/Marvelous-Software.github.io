CalculatePrice

Member resides in the imsAO_PriceMatrixDetail Class

Summary


Gets a unit price based on this price break.

Remarks


This expects PriceMatrix CurrencyMultipliers to be previously set and uses them for the calculation.

Returns


Calculated unit price. It will be converted based on the settings to PriceMatrix.DomesticCurrencyMultiplier And PriceMatrix.PriceMatrixCurrencyMultiplier.
