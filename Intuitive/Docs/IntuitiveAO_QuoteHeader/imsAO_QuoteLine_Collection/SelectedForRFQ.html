SelectedForRFQ

Member resides in the imsAO_QuoteLine_Collection Class

Summary


Return all line records with lQL_AddToThisRFQ updated to true when the line QL_AddToRFQ is true, the line has not been added to a RFQ, and the quantity is greater than zero.

Returns


ArrayList: All quote lines with lQL_AddToThisRFQ updated.
