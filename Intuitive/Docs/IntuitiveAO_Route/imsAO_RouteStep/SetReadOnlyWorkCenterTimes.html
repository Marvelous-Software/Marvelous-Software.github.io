SetReadOnlyWorkCenterTimes

Member resides in the imsAO_RouteStep Class

Summary


When this is an outplant work center disable the fields RTG_StdMachineTime and RTG_StdLabTime. When this is not an outplant work center disable RTG_OutplantVarTime. When the work center is machine loaded disable RTG_StdLabTime. When the work center is labot loaded disable RTG_StdMachineTime. When this is not an outplant work center disable RTG_CostPerUnit.

Remarks


Do not clear the values for the field, just disable them.
