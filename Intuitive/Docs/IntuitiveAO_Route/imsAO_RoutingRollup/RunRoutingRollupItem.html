RunRoutingRollupItem

Member resides in the imsAO_RoutingRollup Class

Summary


Apply route costs to item costs.

Remarks


The default values are provided for the other parameters. The Item Costs are only updated when the master control for Routing Rollup Flag is enabled.

Returns


imsReturnData: When the stored procedure is not successful MSG.RoutingRollupUnsuccessful is added to r2d2.
