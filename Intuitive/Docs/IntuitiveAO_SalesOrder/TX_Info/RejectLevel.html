RejectLevel

Member resides in the TX_Info Class

Summary


Level at which to reject. While the setting is hard to understand, the definition per the legacy system has be retained to confuse old and young alike: "" Reject the SO if an error occurs at any level. "line item" Skip line items and deliveries that have errors; do not reject the entire SO. "delivery" Skip the deliveries that have errors; do not reject the entire SO. Note that the value returned will be all lower case.

Returns


"", "line item", "delivery"
