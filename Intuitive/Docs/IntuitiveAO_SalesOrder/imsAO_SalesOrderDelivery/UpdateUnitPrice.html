UpdateUnitPrice

Member resides in the imsAO_SalesOrderDelivery Class

Summary


Updates SOD_UnitPrice to the price taken from the price matrix or the price passed as a parameter.

Remarks


Do not pass the stdPrice as NewPrice, or else SalesOrder.DontAllowPricingAutomation will be ignored. If you have already queried stdPrice and want to pass it to keep price matrix from being requeried, then you must check SalesOrder.DontAllowPricingAutomation yourself.

Returns


return data
