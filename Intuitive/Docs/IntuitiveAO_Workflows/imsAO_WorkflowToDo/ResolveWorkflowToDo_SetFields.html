ResolveWorkflowToDo_SetFields

Member resides in the imsAO_WorkflowToDo Class

Summary


Sets the fields involved when resolving the to do.

Remarks


The function called be called by an instance of the appobject that has already been loaded. Note that is sets the ResolveWfFlag which will cause an event to be raised to the workflow if one is indicated by the workflow ID field.
