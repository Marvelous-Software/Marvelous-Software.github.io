Member resides in the imsUI_BOMViewer Class

Summary


Display single or multi-level bill of material

Remarks


If ATP Engine object is passed in, then listviews display the ATP quantity for each item. Whether or not the atp horizon is in force is controlled by the caller by setting the enforce horizon property on the engine. Two-way browse mode is the default if no params are passed in. Pass in params to cause single or multi-level bom or whereused.
