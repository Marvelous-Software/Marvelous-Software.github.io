Intuitive ERP is a software package that was developedto Aptean which is the current owner.

I used this software for a company that made helicopter parts. An interesting concept about this software is most all the business objects and forms are inheritable. So if you have a business requirement not addressed with the system all you need to do is inherit the appropriate object and add your changes. Not all functions are designed to be overridden though and if you replace a function you run the risk of not getting updates when a hot fix (patch) is released.


I created a forum to try to get any developers working on this software together to share code although I didn't get a large response.
