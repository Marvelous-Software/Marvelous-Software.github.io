Updating a repository.


At the command prompt of the working folder
1) git add . (adds all changed files)
2) git commit (marks changed files to e updated.)
3) git push -u origin master (copies data to github.)

Creating a new repository from an existing project.


At the Github site.
1) Create repository without a read me otherwise an error will occur.
At the command prompt
2) git init (one time command which creates a .git sub folder and a master branch)
3) optional, create a read me locally.
4) optional, create a .gitignore or copy one to your working directory.
5) git add . (adds all files)
6) git commit -m "x" (x is a commit comment.)
7) git remote add origin https://github.com/{LoginID}/{RepositoryName}.git OR git remote add origin https://github.com/Marvelous-Software/Code-Generator.git
8) git push -u origin master
